# -Bank-Loan-Analysis-Project-SQL-SERVER-MANAGEMENT-STUDIO-Power-BI-Excel
Bank Loan Analysis project using SQL Server, Power BI, and Excel to analyze loan data, assess risk, and identify approval trends. Includes data cleaning, transformation, KPI tracking, and interactive dashboards for data-driven decision-making.

Objectives
Analyze loan approval and rejection patterns
Identify high-risk and low-risk customers
Track key performance indicators (KPIs)
Understand customer demographics and behavior
Support data-driven decision-making
🛠️ Tools & Technologies
SQL Server Management Studio (SSMS) – Data extraction & querying
Power BI – Data visualization & dashboard creation
Excel – Data cleaning & preprocessing
📊 Key Analysis Performed
Data Cleaning & Transformation
Handling Missing Values
SQL Queries for Data Extraction
KPI Calculation (Total Loans, Approved, Rejected, Default Rate)
Customer Segmentation
Risk Analysis
Trend Analysis (Time-based, Region-wise)
📈 Dashboard Features
Loan Approval vs Rejection Analysis
Default Risk Overview
Customer Demographics (Age, Income, Region)
Loan Amount Distribution
KPI Summary Cards
📂 Dataset Description

The dataset includes the following fields:

Loan ID
Customer ID
Age
Income
Loan Amount
Loan Status (Approved/Rejected)
Credit Score
Employment Status
Region
🔍 Key Insights
Identified factors influencing loan approval
Highlighted high-risk customer segments
Found trends in loan distribution across regions
Improved understanding of default patterns
🚀 How to Use
Run SQL queries in SSMS to extract data
Clean and preprocess data in Excel
Load data into Power BI
Explore dashboards and insights
